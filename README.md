# Rainfall_Prediction
## Dataset Description
The dataset consists of 366 daily weather records, representing one full year of observations. Each record contains 12 meteorological attributes related to temperature, atmospheric conditions, and precipitation.
------------------------------------------------------------
## Dataset Overview
  Total records: 366
  Total features: 12
  Data types: Numerical (int and float) and categorical
  Memory usage: ~34.4 KB
------------------------------------------------------------
## Data Quality Notes:

The dataset is mostly complete, with only one missing value each in winddirection and windspeed.

The rainfall column is categorical and requires encoding for machine learning models.

All other features are numerical and suitable for direct modeling.

Intended Use

This dataset is suitable for:

Weather pattern analysis

Rainfall prediction

Regression and classification models

Time-series and seasonal trend analysis
